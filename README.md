•	What is the difference between var, let, and const?

•	4️⃣ What is an arrow function?
•	5️⃣ What are template literals?

Ans. 1: Var is older way to declare variables. Now a days, it is almost not used. The value with var can be redeclared and reassigned. Var shows Hoisted (initialized with undefined) nature. Var is function scoped (or global).
	Let is commonly used to declare variables where values need to be reassigned. It is block scoped (scoped to {}). Let does not require initialization.
	Const can’t be redeclared or reassigned. It is block scoped and must be initialized when declared.

Ans. 2: The spread operator is used to expand elements of any iterable(array, string or object). Spread Operator is denoted by three dots(…). It performs copying arrays or objects. It can combine multiple arrays or objects into a single new one.

Ans. 3: map(), filter(), and forEach() are all JavaScript array methods. The main differences are what they return and when to use. 
forEach is used for executing an action for each element and returns nothing.
Map() is used to convert each element and return a new array.
Filtet() is used for selecting a subset of elements that meet a specific condition and returns a new array.

Ans. 4: Arrow function is commonly used function expression in JavaScript using the => (arrow) operator. It provides a shorter version to traditional function expression. Arrow function returns values implicitly in single line statement. It is commonly used for callbacks.

Ans. 5: In JavaScript, Template literals are introduced in ES6. It allows embedded expressions, multiline strings, dynamically data load. Syntax for Template literals is backticks(`) instead of quotes(“). Template literals make the code cleaner and easier to read.
